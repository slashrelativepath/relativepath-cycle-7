# relativepath cycle7

A set of scripts to manage different systems.

## warning:
This script will install Choco, Nano, Git, Multipass, and VirtualBox(windows)

## usage

On Windows:

`Set-ExecutionPolicy Bypass -Scope Process -Force; .\local-windows.ps1`

On Linux:

`bash local-linux.sh`

On Darwin(Mac):

`zsh local-darwin.sh`
